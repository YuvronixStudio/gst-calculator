## GST Calculator

A simple, fast, and accurate GST (Goods and Services Tax) calculator built using
HTML, CSS, and Vanilla JavaScript.

Designed for small businesses, freelancers, students, and bloggers.
No frameworks. No dependencies. Works offline.

--------------------------------------------------
SETUP & USAGE
--------------------------------------------------

1. Open index.html in any modern web browser
2. Upload the folder to Netlify / GitHub Pages for a live demo
3. Enter an amount and GST rate, or use preset GST buttons
4. Choose GST type: Add GST or Remove GST
5. Results update instantly as you type

--------------------------------------------------
CUSTOMIZATION (5 MINUTES)
--------------------------------------------------

• Change currency symbol  
  Open style.css and edit:
  --currency: "₹";

• Change default GST rate  
  Open index.html and change the value of:
  id="rate"

• Change colors or branding  
  Edit the :root variables in style.css
